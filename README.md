# 📚 Miniguia de Estudos com NotebookLM

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio prático da DIO com foco no uso da Inteligência Artificial como ferramenta de aprendizagem ativa.
A proposta consiste na criação de um caderno temático utilizando o NotebookLM, reunindo fontes abertas sobre um tema financeiro introdutório, aplicando técnicas de engenharia de prompts e organizando o conhecimento em um miniguia de estudos.

---

# 🎯 Tema Escolhido

> Investimentos para iniciantes: entendendo renda fixa e FIIs
---
# 🧠 Objetivos de Estudo

Os principais objetivos deste estudo foram:

- Compreender os conceitos fundamentais sobre o tema.
- Explorar o uso do NotebookLM como ferramenta de apoio aos estudos
- Aplicar técnicas de engenharia de prompt
- Desenvolver pensamento crítico na interação com IA
- Criar um material reutilizável para futuras revisões

---

# ⚠️ Troubleshooting e "Cicatrizes"

Durante os testes, algumas dificuldades foram identificadas:

- Respostas muito genéricas
- Falta de profundidade em determinados conceitos
- Necessidade de adicionar mais contexto aos prompts
- Necessidade de especificar melhor o formato de saída
- Necessidade de validar algumas informações nas fontes originais

## ✅ Soluções Aplicadas

- Uso de prompts mais específicos
- Inclusão de exemplos nos prompts
- Divisão das tarefas em etapas menores
- Definição clara do formato de saída
- Refinamento iterativo dos prompts
